---
title: "Listar produtos"
api: "GET /api/public/infoproducts/v1/products"
description: "Lista os produtos digitais do workspace de Infoprodutos."
---

Retorna os produtos do Builder vinculados ao dono da API Key.

## Query params

<ParamField query="search" type="string">
  Filtra produtos pelo nome ou descricao.
</ParamField>

## Exemplo

```bash
curl --request GET \
  --url "https://api.nerix.com.br/api/public/infoproducts/v1/products?search=curso" \
  --header "Authorization: Bearer nerix_builder_SUA_CHAVE"
```
