---
title: "Obter produto"
api: "GET /api/public/infoproducts/v1/products/{productId}"
description: "Consulta um produto do Builder por ID interno ou referencia publica."
---

## Path params

<ParamField path="productId" type="string" required>
  ID numerico do produto ou referencia publica reconhecida pela plataforma.
</ParamField>

## Exemplo

```bash
curl --request GET \
  --url "https://api.nerix.com.br/api/public/infoproducts/v1/products/4146753" \
  --header "Authorization: Bearer nerix_builder_SUA_CHAVE"
```

## Resposta

```json
{
  "id": 4146753,
  "name": "+200 Mapas Mentais para aprender Frances",
  "price": "10.00",
  "is_active": true,
  "builder_settings": {
    "collect_phone": false,
    "phone_required": false,
    "collect_address": false,
    "address_required": false,
    "require_email_confirmation": false,
    "show_nerix_branding": true
  },
  "offers": [],
  "checkout_links": []
}
```

## Produto nao encontrado

```json
{
  "error": "Produto nao encontrado"
}
```
