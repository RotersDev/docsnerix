---
title: "Obter pedido"
api: "GET /api/public/infoproducts/v1/orders/{orderId}"
description: "Consulta um pedido por ID interno ou numero do pedido."
---

## Path params

<ParamField path="orderId" type="string" required>
  ID numerico do pedido ou `order_number`.
</ParamField>

## Exemplo

```bash
curl --request GET \
  --url "https://api.nerix.com.br/api/public/infoproducts/v1/orders/019f3cbc-3db5-7445-b0f0-598baa2af6a0" \
  --header "Authorization: Bearer nerix_builder_SUA_CHAVE"
```
