---
title: "Listar clientes"
api: "GET /api/public/infoproducts/v1/customers"
description: "Lista compradores consolidados por e-mail."
---

## Exemplo

```bash
curl --request GET \
  --url "https://api.nerix.com.br/api/public/infoproducts/v1/customers" \
  --header "Authorization: Bearer nerix_builder_SUA_CHAVE"
```

## Resposta

```json
{
  "data": [
    {
      "customer_email": "cliente@exemplo.com",
      "customer_name": "Cliente Teste",
      "customer_phone": "41999999999",
      "total_orders": "2",
      "total_paid": "194.00"
    }
  ]
}
```
