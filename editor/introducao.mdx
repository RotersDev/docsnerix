O **Editor de Códigos da Nerix** é uma ferramenta poderosa desenvolvida para designers e desenvolvedores que desejam total controle criativo sobre a interface de suas lojas. 

Diferente de outras plataformas com templates rígidos, a Nerix permite editar diretamente o código-fonte HTML, CSS e JavaScript do seu tema, além de utilizar a nossa engine de renderização para puxar dados dinamicamente.

## 🚀 Como Funciona a Engine de Temas?

A nossa engine processa arquivos HTML estáticos e insere blocos dinâmicos baseados no sistema de templates. Você pode:
1. Ler e exibir **variáveis globais** da loja (ex.: nome, logo, produtos no carrinho).
2. Usar **tags de controle** para loops de repetição (`for`) e condicionais (`if`).
3. Aplicar **filtros** para formatar preços, datas e strings de maneira simples.
4. Definir configurações personalizáveis pelo painel administrativo através do arquivo `schema.json`.

---

## 💡 Primeiros Passos

Para começar a editar o tema de sua loja:
1. Acesse o Painel da Nerix.
2. Vá em **Aparência → Editor de Código**.
3. Selecione o arquivo que deseja editar no menu lateral esquerdo ou configure o nosso [Editor Local (CLI)](/editor/editor-local) para codificar na sua máquina.
