---
title: "Listar Afiliados"
api: "GET /api/public/affiliates"
description: "Retorna a listagem de afiliados aprovados e ativos na sua loja."
---

Retorna os afiliados associados à sua loja, contendo a taxa de comissão individual e dados do usuário.

## Parâmetros de Query

<ParamField query="page" default="1" type="number">
  Página de resultados.
</ParamField>

<ParamField query="limit" default="15" type="number">
  Quantidade de registros por página.
</ParamField>

## Exemplo de Requisição

```bash
curl --request GET \
  --url 'https://api.nerix.com.br/api/public/affiliates?page=1&limit=15' \
  --header 'X-nerixkey: SUA_CHAVE_AQUI'
```
