---
title: "Listar Avaliações"
api: "GET /api/public/reviews"
description: "Retorna a listagem de avaliações dos clientes da sua loja."
---

Retorna todas as avaliações de produtos (comentários, notas, status de aprovação) cadastradas em sua loja.

## Exemplo de Requisição

```bash
curl --request GET \
  --url 'https://api.nerix.com.br/api/public/reviews' \
  --header 'X-nerixkey: SUA_CHAVE_AQUI'
```
