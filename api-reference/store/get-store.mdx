## Descrição

Retorna os metadados e configurações ativas da loja, como nome, logotipo, tema de cores, integrações ativas (Discord, Telegram) e configurações de gateway de pagamento.

Este endpoint é **totalmente público** — não requer API Key.

## Parâmetros de Query

<ParamField query="subdomain" type="string">
  Slug/subdomínio da loja (ex.: `minha-loja`). Se não informado, a loja é detectada automaticamente pelo domínio da requisição.
</ParamField>

## Exemplo de Requisição

<CodeGroup>

```bash cURL
curl --request GET \
  --url 'https://api.nerix.com.br/api/public/store'
```

```javascript JavaScript
const response = await fetch('https://api.nerix.com.br/api/public/store');
const store = await response.json();
console.log(store.name);
```

```python Python
import requests

response = requests.get('https://api.nerix.com.br/api/public/store')
store = response.json()
print(store['name'])
```

</CodeGroup>
