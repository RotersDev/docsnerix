---
title: "Listar Categorias"
api: "GET /api/public/categories"
description: "Retorna as categorias ativas da sua loja."
---

Retorna uma lista de categorias da loja, ordenadas pela ordem de exibição definida no painel administrativo.

## Parâmetros de Query

<ParamField query="include_inactive" default="false" type="boolean">
  Se `true`, inclui categorias inativas na listagem. **Observação: Requer autenticação via API Key para consultar categorias inativas.**
</ParamField>

## Exemplo de Requisição

```bash
curl --request GET \
  --url 'https://api.nerix.com.br/api/public/categories' \
  --header 'X-nerixkey: SUA_CHAVE_AQUI'
```
