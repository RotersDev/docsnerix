---
title: "Listar Clientes"
api: "GET /api/public/customers"
description: "Retorna a listagem de clientes cadastrados na sua loja."
---

Retorna os clientes que já realizaram cadastro ou compras em sua loja associada à API Key.

## Parâmetros de Query

<ParamField query="email" type="string">
  Filtra clientes cujo e-mail corresponde parcialmente ao termo de busca.
</ParamField>

<ParamField query="page" default="1" type="number">
  Página de resultados a ser exibida.
</ParamField>

<ParamField query="limit" default="15" type="number">
  Quantidade de itens por página.
</ParamField>

## Exemplo de Requisição

```bash
curl --request GET \
  --url 'https://api.nerix.com.br/api/public/customers?page=1&limit=15' \
  --header 'X-nerixkey: SUA_CHAVE_AQUI'
```
