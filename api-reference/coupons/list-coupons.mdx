---
title: "Listar Cupons"
api: "GET /api/public/coupons"
description: "Retorna todos os cupons cadastrados na sua loja."
---

Retorna a lista completa de cupons ativos e inativos configurados na loja.

## Exemplo de Requisição

```bash
curl --request GET \
  --url 'https://api.nerix.com.br/api/public/coupons' \
  --header 'X-nerixkey: SUA_CHAVE_AQUI'
```
