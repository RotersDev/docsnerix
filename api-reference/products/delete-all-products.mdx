---
title: "Excluir Todos os Produtos"
api: "DELETE /api/public/products/all"
description: "Remove todos os produtos da loja permanentemente."
---

Realiza a exclusão em massa de todos os produtos cadastrados na loja.

<Danger>
  Esta ação é **irreversível**. Todos os produtos, variantes e chaves de estoque serão permanentemente removidos. Use com extremo cuidado.
</Danger>

## Exemplo de Requisição

```bash
curl --request DELETE \
  --url 'https://api.nerix.com.br/api/public/products/all' \
  --header 'X-nerixkey: SUA_CHAVE_AQUI'
```

## Exemplo de Resposta

```json
{
  "deleted": 128,
  "message": "Todos os produtos foram excluídos com sucesso."
}
```
