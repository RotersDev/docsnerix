---
title: "Buscar Produto"
api: "GET /api/public/products/{id}"
description: "Busca os detalhes de um produto específico através do ID."
---

Busca um produto individual informando seu ID sequencial. Retorna todos os dados do produto, incluindo variantes, imagens e estoque atual.

## Parâmetros de Path

<ParamField path="id" type="number" required>
  ID numérico único do produto (ex.: `12`).
</ParamField>

## Exemplo de Requisição

```bash
curl --request GET \
  --url 'https://api.nerix.com.br/api/public/products/12' \
  --header 'X-nerixkey: SUA_CHAVE_AQUI'
```

{

## Códigos de Erro

| Código | Descrição |
|--------|-----------|
| `404` | Produto não encontrado para o ID informado. |
| `401` | API Key inválida ou ausente (para produtos inativos). |
